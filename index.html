LAMP

发表于 2020-06-17

centos6.9基于FPM模式的编译安装实现LAMP应用

需要的软件包

  1. mairadb：通用二进制格式，mariadb-10.2.12.tar.bz2

  2. httpd：编译安装，httpd-2.4.29.tar.bz2

  3. php7：编译安装，php-7.2.1.tar.bz2

  4. Xcache：编译安装xcache-3.2.0

  5. APR： 下载APR 1.6.3源码

  6. APR-util：下载APR-util 1.6.1源码

  7. phpMyAdmin：下载 phpMyAdmin-4.4.15.10-all-languages

  8. wordpress: 下载 wordpress-4.9.1-zh_CN.tar.gz

顺序：mariadb–>httpd–>php

mariadb官网：https://mariadb.org/

php官网：https://www.php.net/

http官网：http://httpd.apache.org/

安装步骤：

[root@centos6 ~]#mkdir src
[root@centos6 ~]#mv *.bz2 *.gz src
[root@centos6 ~]#cd src
[root@centos6 src]#tar xvf httpd-2.4.29.tar.bz2
[root@centos6 src]#tar xvf apr-1.6.3.tar.bz2
[root@centos6 src]#tar xvf apr-util-1.6.1.tar.bz2

[root@centos6 src]#mv apr-1.6.3 httpd-2.4.29/srclib/apr
[root@centos6 src]#mv apr-util-1.6.1 httpd-2.4.29/srclib/apr-util
[root@centos6 src]#cd httpd-2.4.29/
[root@centos6 http-2.4.29]#./configure –prefix=/app/httpd24
–enable-so
–enable-ssl
–enable-cgi
–enable-rewrite
–with-zlib
–with-pcre \
–with-included-apr
–enable-modules=most
–enable-mpms-shared
